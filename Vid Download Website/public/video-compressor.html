How it works

We use FFmpeg with CRF (Constant Rate Factor) to intelligently reduce file size. It analyzes each frame and allocates bits where they matter — keeping visuals clear while shrinking the file.

Which mode to pick?

Target MB — set an exact size limit. Reduce % — shrink by a percentage of the original. Manual CRF — fine control: lower CRF = better quality, higher CRF = smaller file.

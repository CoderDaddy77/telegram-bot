How it works

Video compression runs entirely in your browser using WebAssembly (FFmpeg.wasm). Your file never leaves your device — zero upload, zero server processing. Privacy first.

Which mode to pick?

Target MB — set an exact size limit. Reduce % — shrink by a percentage of the original. Manual CRF — fine control: lower CRF = better quality, higher CRF = smaller file.

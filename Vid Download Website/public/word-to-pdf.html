How it works

We parse your DOCX file in the browser using mammoth.js, render it to styled HTML, then convert to PDF using jsPDF. No server, no upload.

What's supported

Text, headings, bold, italic, underline, lists, tables, font sizes and basic styling. Complex layouts, embedded charts, and macros are not supported.

How it works

We parse the PPTX file in your browser using JSZip. Each slide is rendered to a canvas and assembled into a PDF. No server processing.

Limitations

Supports text, basic shapes, images and solid backgrounds. Complex animations, charts, SmartArt and embedded videos are not fully supported.
